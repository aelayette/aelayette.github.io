# Birdgame Project Images

Place real screenshots and recordings here. Recommended filenames (referenced in projects/birdgame.html):

- `level-overview.png` — Wide shot of the level environment
- `gameplay.png` — Player character in action
- `blueprint-editor.png` — Blueprint graph or UE5 editor view
- `lighting.png` — Lighting / atmosphere pass

Image format: PNG or JPG, 16:9 aspect ratio preferred.
